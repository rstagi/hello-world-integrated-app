## 🔧 Development

To run locally:

```bash
# Clone the repository
git clone https://github.com/rstagi/hello-world-integrated-app.git
cd hello-world-integrated-app

# Start local server
npm start
# or
python3 -m http.server 8000
```

Visit `http://localhost:8000` to view the app.
